# Alfido Internship - Task 2

## Algorithms & Problem Solving (Sorting / Searching / Greedy)

## Objective

Implement algorithmic problems demonstrating sorting, searching, and greedy approaches using C++.

## Implemented Programs

### 1. Binary Search

Searches for an element in a sorted array.

Operations:

* Divide search space into halves
* Compare middle element
* Return index if found

Time Complexity:

* Best Case: O(1)
* Worst Case: O(log n)

---

### 2. Merge Sort

Sorts array using divide and conquer approach.

Operations:

* Divide array into subarrays
* Merge sorted subarrays

Time Complexity:

* Best Case: O(n log n)
* Worst Case: O(n log n)

---

### 3. Knapsack Problem (Greedy)

Find maximum profit within limited capacity.

Operations:

* Calculate profit/weight ratio
* Select items based on capacity

Time Complexity:

* O(n)

---
