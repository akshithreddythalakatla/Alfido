# Data Structures Implementation in C++

## Implemented

1. Stack
2. Linked List

## Operations

### Stack
- Push
- Pop
- Search
- Display

### Linked List
- Insert
- Delete
- Search
- Display

## Compile Commands

g++ stack.cpp -o stack
./stack

g++ linkedlist.cpp -o linkedlist
./linkedlist

## Time Complexity

Stack:
Push - O(1)
Pop - O(1)
Search - O(n)

Linked List:
Insert - O(1)
Delete - O(n)
Search - O(n)
